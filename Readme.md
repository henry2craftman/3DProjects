# Guardian's Valor (가디언의 용맹)

## 🎮 프로젝트 개요

**Guardian's Valor**는 Unity 엔진으로 개발된 3D 타워 디펜스 게임입니다. 플레이어는 전략적으로 타워를 건설하여 몰려오는 적들로부터 기지를 방어해야 합니다. 이 프로젝트는 '타워 디펜스 강의'를 기반으로 제작되었으며, 게임의 핵심 메커니즘과 확장 가능한 아키텍처를 구현하는 데 중점을 두었습니다.

---

## ✨ 주요 기능

- **웨이브 기반 적 스폰:** `ObjectPool`을 사용하여 적들을 주기적으로 생성하고 비활성화하여 메모리 사용을 최적화합니다.
- **타워 건설 및 경제 시스템:**
    - 플레이어는 `Bank`에 저장된 골드를 사용하여 타워를 건설할 수 있습니다.
    - 타워 건설 시 비용이 차감되며, 잔액이 부족하면 건설할 수 없습니다.
- **경로 시스템:** 적들은 사전에 정의된 `Waypoint`를 따라 이동합니다.
- **게임 오버 조건:** 적이 최종 목적지에 도달하면 플레이어의 골드가 감소하며, 골드가 0 미만이 되면 게임이 다시 시작됩니다.

## 🛠️ 기술적 특징 및 해자 (Technical Moat)

이 프로젝트는 단순한 기능 구현을 넘어, 다음과 같은 기술적 특징을 통해 확장성과 성능을 확보했습니다.

### 1. 오브젝트 풀링 (Object Pooling)
- 적(Enemy) 오브젝트를 반복적으로 생성(`Instantiate`)하고 파괴(`Destroy`)하는 대신, **오브젝트 풀**을 구현하여 재사용합니다.
- 이를 통해 게임 중 발생하는 가비지 컬렉션(GC)을 최소화하고, 특히 많은 수의 적이 등장하는 웨이브에서 프레임 드랍을 방지하여 부드러운 게임 플레이를 보장합니다.

### 2. 동적 경로 탐색을 위한 그리드 시스템 기반
- `GridManager`는 게임 월드를 2D 격자(Grid)로 나누고, 각 좌표에 해당하는 `Node` 정보를 `Dictionary` 자료구조로 관리합니다.
- `Awake()` 시점에서 각 `Node`는 `isWalkable = true` 상태로 초기화되며, 이는 향후 경로 탐색 알고리즘 적용을 위한 기본 데이터 구조입니다.
- **현재** 적의 이동은 사전 정의된 `Waypoint`를 따르지만, 이 그리드 시스템은 **A\* (A-Star)와 같은 고급 경로 탐색 알고리즘**을 도입할 수 있는 핵심적인 기반(Foundation) 역할을 합니다.
- 이를 확장하면, 플레이어가 타워를 설치하여 길을 막으면 적이 동적으로 경로를 재계산하는 등 한 차원 높은 전략적 게임 플레이를 구현할 수 있습니다.

### 3. 고급 터레인 셰이딩 (MicroSplat)
- Unity Asset인 **MicroSplat**을 프로젝트에 통합하여, 표준 터레인 셰이더보다 훨씬 뛰어난 비주얼 퀄리티를 구현했습니다.
- 이를 통해 복잡한 텍스처 블렌딩, 높이 맵 기반의 디테일 표현 등이 가능하여, 게임의 시각적 완성도를 크게 향상시켰습니다.

### 4. 컴포넌트 기반의 유연한 아키텍처
- `Bank`, `Tower`, `EnemyMover` 등 각 기능별로 역할을 명확히 분리한 컴포넌트 기반으로 설계되었습니다.
- 이는 코드의 재사용성을 높이고, 기능 추가 및 수정 시 다른 부분에 미치는 영향을 최소화하여 유지보수가 용이한 구조를 만듭니다.

## 🚀 실행 방법

1. 이 프로젝트를 클론하거나 다운로드합니다.
2. Unity Hub를 엽니다.
3. 'Add project from disk'를 선택하고 이 프로젝트 폴더를 지정합니다.
4. 프로젝트에 맞는 Unity Editor 버전(권장: `2021.3` 이상)을 선택하여 엽니다.
5. `Assets/Guardian's Valor` 폴더에 있는 `Guardian's Valor.unity` 씬을 열어 게임을 실행합니다.
